# 1. Product Overview

## Product Name
**Rifky Portfolio**

## Purpose
Membangun website portofolio pribadi yang menampilkan karya, layanan, dan informasi profesional dengan tampilan yang elegan, modern, dan minimalis.

Website harus mampu memberikan kesan tenang, premium, natural, dan profesional tanpa terlihat ramai.

Target utama bukan sekadar menunjukkan project, tetapi membangun personal branding.

# 2. Goals

## Primary Goals
- Meningkatkan personal branding
- Menampilkan project terbaik
- Menawarkan jasa
- Memudahkan calon client menghubungi
- Menjadi media showcase skill

## Success Metrics
- Loading cepat (<2 detik)
- Responsive di semua device
- Navigasi mudah dipahami
- Animasi halus tanpa mengganggu user
- UI konsisten di seluruh halaman

# 5. Design Concept

## Theme
Nature Minimalism

Inspirasi:
- Hutan
- Kayu
- Daun
- Batu
- Cahaya matahari
- Kertas daur ulang
- Tekstur alami

## Color Palette

### Primary
- Forest Green `#355E3B`
- Earth Brown `#6B4F3A`

### Neutral
- Warm White `#F7F6F2`
- Charcoal Black `#1E1E1E`
- Soft Gray `#D8D6D1`

## Background
- Matte / Doff
- Sedikit grain
- Sedikit noise texture
- Warna lembut

## Texture
Gunakan texture secara subtle:
- Daun
- Kayu
- Kertas
- Linen
- Grain

Opacity: **5–10%**

# 6. Visual Style

- Banyak whitespace
- Border tipis
- Rounded kecil
- Shadow sangat halus
- Tanpa warna mencolok
- Tanpa gradient neon

# 7. Animation Style

Durasi: **400–800ms**

Easing:
`ease-in-out`

Animasi:
- Fade In
- Slide Up
- Parallax Background
- Image Zoom Hover (scale 1.03)
- Button Hover
- Text Reveal
- Sticky Navbar dengan blur saat scroll
